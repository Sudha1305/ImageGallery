# Image Gallery

A beautiful, modern image gallery featuring travel photography from around the world.

## Features

- **Responsive Design**: Works perfectly on all devices (mobile, tablet, desktop)
- **Dark/Light Theme**: Toggle between dark and light color schemes
- **Search Functionality**: Find images by tags, descriptions, or categories
- **Filtering**: Browse images by category (beaches, temples, wonders, cities)
- **Lightbox View**: Full-screen image view with download capability
- **Trending Tags**: Highlight popular images
- **Touch Support**: Zoom and pan with touch gestures
- **Keyboard Navigation**: Navigate with arrow keys and close with Escape

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Or start a local server for best experience:

```bash
# Python
python -m http.server 8000

# Then visit http://localhost:8000
```
